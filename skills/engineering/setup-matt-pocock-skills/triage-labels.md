# 分流 标签

技能使用五个规范 triage 角色来表述. 此文件将这些角色映射到本仓库 issue tracker 中实际使用的 label 字符串.

| mattpoclock/skills 中的 Label | 我们 tracker 中的 Label | 含义                                      |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此 issue                      |
| `needs-info`               | `needs-info`         | 等待报告者提供更多信息                      |
| `ready-for-agent`          | `ready-for-agent`    | 完全明确, AFK agent 可拿取                 |
| `ready-for-human`          | `ready-for-human`    | 需要人类实现                                |
| `wontfix`                  | `wontfix`            | 不会采取行动                                |

当技能提及一个角色 (例如 "应用 AFK-ready triage label"), 使用此表中对应的 label 字符串.
