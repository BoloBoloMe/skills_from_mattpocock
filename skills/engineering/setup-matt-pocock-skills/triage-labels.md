# 分流（Triage）标签

这些技能使用五个标准分流（triage）角色来表达状态。本文件将这些角色映射到此仓库 issue tracker 中实际使用的 label 字符串。

| mattpocock/skills 中的 label | 我们跟踪器中的 label | 含义                              |
| ---------------------------- | -------------------- | --------------------------------- |
| `needs-triage`               | `needs-triage`       | 维护者需要评估此 issue            |
| `needs-info`                 | `needs-info`         | 等待报告者提供更多信息            |
| `ready-for-agent`            | `ready-for-agent`    | 已完整说明，适合 AFK agent 接手   |
| `ready-for-human`            | `ready-for-human`    | 需要人类实施                      |
| `wontfix`                    | `wontfix`            | 不会处理                          |

当某个技能提到角色时（例如“应用适合 AFK 的分流标签（AFK-ready triage label）”），使用此表中的对应 label 字符串。
