# 进行中

仍在开发中的 skills。它们还没准备好发布——可能会有粗糙边缘、破坏性变更和被放弃的实验。在它们毕业进入稳定分类（bucket）之前，会被排除在插件（plugin）和顶层 README 之外。

- **[review](./review/SKILL.md)** — 从固定点之后，沿两条并行轴审查变更：**标准（Standards）**（diff 是否遵循仓库的编码标准？）和 **规格（Spec）**（diff 是否忠实实现了来源 issue/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 以节拍（beats）旅程的方式塑造文章，类似选择你自己的冒险。选择一个起始节拍（beat），只写那个节拍，然后转向下一个，直到文章自然结束。
- **[writing-fragments](./writing-fragments/SKILL.md)** — 一场追问会话（grilling session），从你那里挖掘片段（fragments）——异质的写作片段——并把它们作为未来文章的原材料追加到单个文档中。
- **[writing-shape](./writing-shape/SKILL.md)** — 取一个包含原材料的 Markdown 文件，逐段把它塑造成一篇文章，并在每一步论证格式选择。
